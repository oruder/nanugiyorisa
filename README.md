# 나누기 요리사 (Division Chef)

초등학교 4학년 학생들을 위한 나누기 학습 프로그램

## 🎮 게임 방법

1. **주문 확인하기**
   - 상단의 주문서에서 필요한 재료와 양을 확인하세요
   - 분수로 표시된 양을 잘 보세요!

2. **재료 가져오기**
   - 왼쪽 재료 창고에서 원하는 재료를 클릭하여 작업대로 가져오세요
   - 여러 개 가져올 수 있어요

3. **재료 나누기**
   - 작업대의 재료를 클릭한 후 "등분하기" 버튼을 누르세요
   - 2등분, 3등분, 4등분 등 원하는 만큼 나눌 수 있어요

4. **접시에 담기**
   - 나눠진 조각을 클릭하면 자동으로 접시에 담겨요
   - 필요한 만큼만 담으세요!

5. **서빙하기**
   - 주문한 양과 똑같이 담았다면 "서빙하기" 버튼을 누르세요
   - 정답이면 다음 문제로 넘어가요!

## 🎯 학습 목표

- 분수의 개념 이해하기
- 나눗셈의 원리 학습하기
- 전체와 부분의 관계 파악하기
- 문제 해결 능력 향상하기

## 📚 난이도

- **레벨 1**: 기초 (1/2, 1/4, 3/4 등 간단한 분수)
- **레벨 2**: 중급 (1/3, 2/3, 1/6 등 3등분과 6등분)
- **레벨 3**: 고급 (3/8, 5/12 등 복잡한 분수와 여러 재료)

## 🚀 시작하기

`index.html` 파일을 웹 브라우저에서 열면 바로 게임을 시작할 수 있습니다!
